Course Description

The main objective of this course is to introduce the basic concepts of object-oriented programming, using a pure object-oriented language like Java. UML will be presented succinctly to understand the modeling approach by objects. Some useful static diagrams such as object and class diagrams will also be covered. A complete project illustrating the different approaches of this course will be presented as a model for future projects.
